## Sobre o Projeto

## O que é?

É um website feito a partir de um minimundo em html css e js.
Para a segunda parte do trabalho, foi pedido uma implementação de um banco de dados e um CRUD.


## Funcionalidades

- Navegação SPA utilizando Hash Routing
- Cadastro de produtos
- Listagem de produtos
- Atualização de registros
- Exclusão de registros
- Integração com banco de dados MySQL

## Stack

*   **Frontend (SPA):** Single Page Application construída com **Vanilla JavaScript** utilizando **Hash Routing (`#`)**.
*   **Backend (API):** Desenvolvido em **PHP**, atuando estritamente como uma API de dados que se comunica via JSON.
*   **Banco de Dados:** **MySQL**, com persistência de dados estruturada e conexões seguras via **PDO (PHP Data Objects)**.
*   **Design & UI/UX:** Interface totalmente planejada e prototipada através do **Figma** antes do desenvolvimento do código.
